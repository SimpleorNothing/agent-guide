5. 1줄/2줄 판정 (양방향 1.5배 기준)

총 환산 자수가 레벨別 안전 상한의 1.5배를 초과하면 2줄 허용: L1 >45 / L2 >42 / L3 >41 → 2줄, 그 외 → 1줄로 압축.
2줄 작성 자체는 정상(감점 사유 아님). 감점은 "1줄을 선택했는데 안전선 초과 또는 채움 하한 미달"인 경우에만 적용.

8. 감점 완화 원칙 (모든 평가 공통 적용)

  • 정량 수치 없는 입력자료 → 인과경로 정량기여도 미명시 완화 (정성 등급화時 만점, 정성 비교만時 60%)
  • 단일 출처 입력 → 출처 다양성 감점 면제
  • 시연용 예시 수치 빨간 Italic → 항상 강제 유지
  • 도표 부족 → 현재 감점분의 50%만 적용
  • 한글(영문) 병기 → 원문 소스에서 한글 표기 안 했으면 감점 면제
  • 헤드라인 정량 KPI → 원문에 정량 수치 없으면 감점 면제
  • 시사점 액션 동사 → 방향성 제시(모니터링 강화·검토·대응)로 충분, 구체 KPI 미적용 감점 면제

9. 인과경로 분해 원칙

  • 원문에 경로·비중 명시 → 그대로 표현 (직접/간접 무리한 추가 구분 불요)
  • 원문이 나열식 → 추가 구조화(직접/간접 분해)는 가점 사유
  • 원문 경로분해 있는데 자의적 재구성 → 감점 사유
