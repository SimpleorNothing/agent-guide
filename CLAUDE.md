# 작업 메모 — agent-guide

## 워크플로 선호도

- 작업 브랜치에 push가 끝나면 **항상** PR을 생성하고, 곧바로 **자동 머지**까지 진행한다.
  - 사용자가 매번 요청하지 않아도 기본 동작으로 수행한다. (2026-07-10 사용자 요청: "향후 자동머지되도록 설정")
  - PR base 는 `main`.
  - 머지 방식은 squash.
  - Cloudflare Workers 배포 등 CI가 통과했는지 확인한 뒤 머지한다.
  - 브랜치 보호 룰·CI 실패 등으로 머지가 막히면 그 사실을 보고한다.
  - 머지 후 작업 브랜치는 삭제한다(remote).
